# 二，档案目录权限
### 1.如何查看
ls -l
[类型][拥有者权限，群组权限，其他人权限][连结][拥有者][群组]
### 2.各参数意义
- r : read，对于档案，可读取内容；对于目录，可查看目录结构
- w : write,对于档案，可修改内容；对于目录，可修改目录结构
- x : execute,对于档案，可被系统执行；对于目录，可设为工作目录
### 3.修改参数
- chgrp : 改变档案所属群组，e.g: chgrp users install.log
- chown : 改变档案拥有者,e.g: chown zhu install.log
- chmod : 改变档案权限
	- 数字类型改变 : r(4),w(2),x(1);e.g: chmod 777 install.log
	- 符号类型改变 : u(user),g(group),o(other),a(all);+,-,=;
	e.g: chmod u=rwx,go=rxx install.log
	e.g: chmod a+w install.log
### 4.档案种类
- 正规档案(-)
	- 纯文本
	- 二进制
	- 数据格式文件
- 目录(d)
- 连结档(l)
- 设备与装置文件
	- 区块设备(b)
	- 字符设备(c)
- 资料接口文件(s)
- 数据传送文件(p)
### 5.档案扩展名
便于快速了解档案可能用途
- *.sh : 脚本或批处理文件(scripts)
- *.Z,*.tar,*.tar.gz,*.zip,*.tgz : 压缩文件
- *.html,*.php : 网页相关
### 6.目录结构
/
	/bin : 可执行文件
	/boot : 开机与核心档
	/usr(Unix Sofeware Resource) : 放置软件
	/opt : 第三方协力软件
	/etc : 配置文件
	/var : 系统运作过程相关
	/dev : 设备

# 三，档案目录管理
### 1.目录操作
###### 特殊目录
- . 此层目录
- .. 上一层目录
- - 前一个工作目录
- ～ 用户家目录
- cd : 变换目录
- pwd : 显示当前目录
- mkdir : 新建目录
- rmdir : 删除空目录
	- r参数 : 递归删除
	- p参数 : 删除上层空目录
### 2.环境变量$PATH
在执行指令时，系统会依照PATH的设定去每个PATH定义到目录下搜寻同名可执行文件
echo $PATH查看
### 3.文档操作
###### 3.1基本操作
- cp : 复制
	- -a : 完全复制
	- -r : 递归复制
	- -s : 创建符号链接
- rm : 删除
	- -r : 递归删除
	- -i : 互动模式
	- -f : 强制删除
- mv : 移动
###### 3.2查询操作
- which : 查看命令位置
- find : 在指定路径查找文件
- grep : 再指定文件中查找字符串
- find : 文件查找，可设定文件参数
- whereis : 查找指定文件的位置
- locate : 档案部分名称查询
###### 3.3查看操作
- cat : 第一行开始显示文档
- tac : 最后一行开始显示文档
- more : 一页页向下显示
- less : 一页页向下显示，可向前翻页
- nl : 显示行号
- head : 看前面几行,-n
- tail : 看后面几行,-n
- od : 以二进制查看

# 四，用户管理
- groupadd : 创建一个用户组
- useradd : 创建一个用户
- passwd : 用户密码设置
- usermod : 修改用户属性信息
- userdel : 删除用户
- groupdel : 删除指定组

# 五，vim的使用
请使用vimtutor学习
三种状态或模式:
### 命令行模式command mode） 
###### 可使用功能
- 控制屏幕光标的移动，字符、字或行的删除，移动复制某区段
- 进入Insertmode下，或者到 last line mode。
- /work 向下搜索 ?work 向上搜索 
- h 左 j 下 k 上 l 右 0 移动到行首 $ 移动到行尾 
- H 屏幕最上 M 屏幕中央 L 屏幕最下 G 档案最后一行
- gg 移动到档案第一行
- dd 删除光标当前行 ndd 删除n行
- yy 复制当前行 nyy 复制n行 p,P 粘贴	
### 插入模式(Insert mode)
按i键进入插入模式只有在Insertmode下，才可以做文字输入。
按「ESC」键可回到命令行模式。
### 底行模式（last line mode） 将文件保存或退出vi，也可以设置编辑环境，如寻找字符串、列出行号……等。 
          底行模式下，如何离开和保存文件 ? wq保存退出   wq!强制保存退出 q不保存退出  q!强制不保存退出
